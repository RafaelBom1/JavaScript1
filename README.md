# JavaScript1

## Variáveis
- var
- const
- let
- string: caracteres
- array: conjunto de valores

## Operadores
- Soma +
- Subtração -
- Multiplicação *
- Divisão /
- Módulo (resto da divisão) %
- Math: random(), round(), sqrt()

* Atribuição
a = b
a += b (soma e atribui em seguida)
a -= b
a *= b
a /= b
a %= b
Incrementa 1 ++
Subtrai 1 --

* Comparação
- Igual == ou ===
- Diferente !=
- Maior que >
- Maior ou igual a >=
- Menor que <
